## 💼 핵심 역량

### 🛣️ Path Planning (판단)
- 실시간 동적/정적 환경에서의 경로 계획 알고리즘 설계
- Hybrid A*, Reeds-Shepp 등 샘플링 기반 경로 생성
- C++ 최적화를 통한 처리 속도 2배 향상 경험

### ⚙️ Vehicle Control (제어)
- 차량 동역학 기반 제어 로직 설계
- 다중 액추에이터 최적 제어 분배(Control Allocation)
- MATLAB/Simulink + CarSim 시뮬레이션 검증

### 🔗 System Integration (통합)
- ROS 기반 인지-판단-제어 S/W 모듈 통합
- 실제 하드웨어(ERP-42)에서 안정적 구동 경험
